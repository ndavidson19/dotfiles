## Installation

* [neovim 0.9.0+](https://github.com/neovim/neovim)

```console
git clone https://github.com/alexjercan/nvim.dotfiles.git
cd nvim.dotfiles
./install
```

## Other Dependencies

For tmux to show the correct icons in the status bar you will need a [nerd
font](https://www.nerdfonts.com/).
